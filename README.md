LuaXe
=====

Lua target for Haxe language

State: *pre-alpha*
=====
Primarily LuaJIT -> make super-optimized portable Haxe target!

I have some working *fast* (beats V8 and Neko) code otput. Will push soon.

Require Haxe Compiler 3.1, Lua 5.2 or LuaJIT 2
